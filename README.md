## About

I originally created this roadmap to reproduce independently, through self-study and practice, the learning progression of a newly introduced cybersecurity degree at a local university.

This repository reorganizes that curriculum into a public reference. It preserves the original structure of terms, workshops, specialization subjects, certifications, resources, communities, and providers contained in my source spreadsheets.

## Honorable mentions: formal online academic programs

These four programs are taught in Spanish, are fully online, and are recommended primarily for Mexican citizens or people residing in Mexico. They may also be useful to other Spanish speakers, but admission procedures, documentation, official recognition, and professional credentialing are tied to the Mexican education system.

### [UVEG — Bachelor's Degree in Cybersecurity Engineering](https://uveg.edu.mx/info-ing-ciberseg)

My recommendation for people starting from zero who have not yet completed higher education. It is an online engineering degree focused directly on protecting information, systems, networks, and technological infrastructure. It combines programming, networking, cryptography, offensive and defensive security, risk management, legislation, cloud security, and artificial intelligence, plus three specialization blocks.

**Subjects:** Computer Skills, Reading and Writing Workshop, Foreign Language I, Algebra for Engineers, Introduction to Cybersecurity, Ethics and Social Responsibility, Discrete Mathematics, Introduction to Computer Systems, Programming Fundamentals, Networking Fundamentals, Sustainable Development Applied to ICT, Foreign Language II, Calculus for Engineers, Database Fundamentals, Programming Workshop, Computer Architecture and Operating Systems, Foreign Language III, Institutional Elective, Probability and Statistics, Computer Networks, Cryptography and Information Security, Operating System Security, Effective Communication Workshop, Foreign Language IV, Network Security, Cybersecurity Applied to Programming, Ethical Hacking and Penetration Testing, Project Management, Leadership and Decision-Making, Information Security Management, Cybersecurity and Data Protection Legislation, Artificial Intelligence, Entrepreneurship, Specialization I, Cloud Security, Cybersecurity Risk and Vulnerability Management, Team Management, Specialization II, Cybersecurity Project Formulation, Specialization III, and Professional Internship.

### [INFOTEC — Higher University Technician in Cybersecurity](https://www.infotec.mx/tsu_cursos/tsu-en-ciberseguridad/)

My favorite and recommended option for people who already have higher education and mainly want concentrated technical knowledge or a practical specialization. It is a fully online six-term program lasting two years and includes a professional internship. It combines technical support, secure software development, telecommunications, networking, cryptography, cloud technologies, monitoring, risk management, and legal frameworks.

**Subjects:** Oral and Written Communication, Differential and Integral Calculus, Computer Fundamentals, Numerical Methods, Project Management, Probability and Statistics, Databases, Networking Fundamentals, Programming Fundamentals, Operating Systems, Data Structures, Information Technology Service Management, Introduction to Electronics, Security Incident Response, Programming Languages with Security Considerations, Computer Architecture, Mobile Architectures, Server Technology, Wired and Wireless Communications, Information Security Risk Management, Software Engineering, Switching and Routing Networks, Cryptography, Certificate and Access Management, Information and Operations Architecture, Telecommunications Monitoring and Control, Internet and Cloud Technologies, Network Security and Ethical Hacking, Application Security, Emerging Technologies, Ethics and Legal Framework, and Professional Internship.

### [UnADM — Software Development Engineering](https://www.unadmexico.mx/division-de-ciencias-exactas-ingenieria-y-tecnologia/desarrollo-de-software)

Recommended for people seeking deep, generalist foundations in programming, software engineering, databases, operating systems, networks, quality, and project management. It is not a cybersecurity degree, but it provides a broad base for later specialization in application security, security engineering, automation, or secure software research.

**Subjects:** Socioeconomic Context of Mexico, Basic Statistics, Programming Fundamentals, Differential Calculus, Human Development, Research Fundamentals, Object-Oriented Analysis and Design, Operating Systems, Integral Calculus, Administration Fundamentals, Databases, Object-Oriented Programming I, Database Design, Business Modeling, Object-Oriented Programming II, Introduction to Software Engineering, Discrete Mathematics, Operating Systems Programming, Data Structures, Thinking Skills, Software Development Methods and Models, Database Administration, Software Design and Architecture, Object-Oriented Programming III, Organizational Structure, Operations Research, .NET Programming I, Networking Fundamentals, Software Development Metrics (PSP), Strategic Planning, Information Project Management, Engineering Economics, Network Interconnectivity, Team Software Development (TSP), Software Quality Models, .NET Programming II, Information Technology Management, .NET Programming III, Investment Projects, Web Programming I, Software System Testing and Maintenance, Web Programming II, Mobile Programming, Computer Security, Terminal Project I, and Terminal Project II.

### [UnADM — Telematics Engineering](https://www.unadmexico.mx/division-de-ciencias-exactas-ingenieria-y-tecnologia/telematica)

Recommended for people seeking deep, generalist foundations with stronger emphasis on networking, telecommunications, electronics, distributed systems, infrastructure, and information security. It is particularly useful as a base for later specialization in network security, cloud infrastructure, telecommunications, or embedded systems.

**Subjects:** Human Development, Basic Statistics, Programming Fundamentals, Research Fundamentals, Socioeconomic Context of Mexico, Differential Calculus, Administration Fundamentals, Databases, Integral Calculus, Discrete Mathematics, Thinking Skills, Operating Systems, Networking Fundamentals, Leadership and Teamwork, Differential Equations, Data Communications, Basic Electronics, Physics I, Digital Electronics, Computer Law, Physics II, Security I, Home and SME Networks, Computer Resource Administration, Web Programming, Security II, Medium-Sized Networks, Computer Architecture I, Digital Economy, Network Optimization, Digital Systems Communication, Operating System Administration, Computer Architecture II, Control Systems Communication, Network Design and Architecture, Social and Political Environment, Network Administration, IT Auditing, Network Planning, E-Commerce Systems, Mobile Devices, Business Project Formulation, Converged Networks, Distributed Databases, Terminal Project I, and Terminal Project II.
