## Acerca de esta ruta

Originalmente creé esta ruta para reproducir por mi cuenta, mediante estudio y práctica, la progresión de aprendizaje de una nueva carrera de ciberseguridad de una universidad local.

Este repositorio reorganiza ese plan como una referencia pública. Conserva la estructura original de cuatrimestres, talleres, materias de especialización, certificaciones, recursos, comunidades y proveedores incluidos en mis hojas de cálculo.

## Plan de estudios

### Primer ciclo de formación

#### Cuatrimestre 1

- Inglés I
- Fundamentos matemáticos
- Fundamentos de ciberseguridad
- Introducción a sistemas operativos y redes
- Ciberseguridad en sistemas operativos
- Comunicación y habilidades digitales

- Ciberseguridad en redes

## Materias de especialización

| Área 1 | Área 2 | Área 3 |
| --- | --- | --- |
| Hacking de API y microservicios | Ingeniería inversa | Análisis estático y dinámico de binarios |
| Wireless hacking | Desarrollo de exploits | Hacking de sistemas de control industrial (SCADA/OT) |
| OSINT y reconocimiento avanzado | Hacking de kernel y sistemas operativos | Hacking de hardware y firmware |
| Análisis y evasión de defensas de detección | Hacking de bases de datos | Análisis avanzado de malware y evasión |
| Investigación de vulnerabilidades | Guerra cibernética y doctrina de ciberoperaciones | Ataques a la cadena de suministro y de hardware |
| Criptografía y criptoanálisis avanzado | Operaciones de influencia y guerra de información | Análisis forense y atribución de amenazas Estado-nación |
| Cloud security avanzado | Análisis de inteligencia de amenazas | Operaciones Red Team y Blue Team |
